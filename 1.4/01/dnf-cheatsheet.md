| Команда                 | Описание                               |
| ----------------------- | -------------------------------------- |
| `dnf search <keyword>`  | Поиск пакета по имени/описанию         |
| `dnf info <package>`    | Подробная информация о пакете          |
| `dnf install <package>` | Установить пакет и зависимости         |
| `dnf remove <package>`  | Удалить пакет (конфиги остаются)       |
| `dnf update`            | Обновить все пакеты                    |
| `dnf history`           | Показать историю транзакций            |
| `dnf history undo <ID>` | Отменить транзакцию по ID              |
| `dnf provides <file>`   | Найти пакет, которому принадлежит файл |
| `dnf clean all`         | Очистить весь кэш пакетов              |
| `dnf autoremove`        | Удалить ненужные зависимости           |
