Полезные критерии find:
-iname - поиск без учёта регистра; удобно для поиска файлов вроде backup, DOC
-size +N / -size -N - быстрый поиск пустых (-size 0) или больших файлов (+5M, +1G)
-mtime -N / -mtime +N - поиск по дате изменения; полезно для бэкапов, логов, временных файлов
-mmin -N - поиск по минутам; точно находит файлы, созданные «только что»
-type f / -type d - фильтрация: только файлы или только каталоги
-exec - выполнение команды над каждым найденным файлом (копирование, архивация)
-delete - быстрая очистка; использовать только после проверки через -print
! -name ".txt" - исключение файлов по маске; гибкая фильтрация

Полезные флаги grep:
-r - рекурсивный поиск по всем файлам в директории
-i - игнорировать регистр; находит bug, Bug, BUG одной командой
-n - показать номер строки; удобно для быстрого перехода к нужному месту
-l - вывести только имена файлов; экономит время при большом объёме совпадений
-v - инверсия; показывает строки, которые НЕ содержат искомый текст
