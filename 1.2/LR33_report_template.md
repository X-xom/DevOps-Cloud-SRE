## Задание (все подпункты)
### Задание 3.1. Подготовка
Создать `~/link_lab`, подготовить `original.txt`, `source.txt`, `target_dir`.

### Задание 3.2. Эксперимент с символическими ссылками
`ln -s`, сравнить inode, удалить оригинал, наблюдать, восстановить, объяснить.

### Задание 3.3. Эксперимент с жесткими ссылками
`ln` без `-s`, сравнить inode, счетчик ссылок, изменение через ссылку, удалить оригинал, наблюдать.

### Задание 3.4. Ограничения
Попытаться создать hardlink на каталог (зафиксировать ошибку). Создать symlink на каталог. (Опц.) hardlink на другой раздел.

### Задание 3.5. Практическое применение
Создать symlink `~/myconf` на `~/.bashrc` (или другой конфиг) и объяснить пользу.
